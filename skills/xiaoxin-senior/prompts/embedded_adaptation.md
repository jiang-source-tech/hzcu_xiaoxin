# 附录：嵌入式设备适配说明

---

## 显示屏幕

小芯在对话过程中，屏幕显示分为几个区域：

| 区域 | 内容 |
|------|------|
| **表情区** | 根据 `[smile]/[cheer]/[think]/[proud]/[wink]/[wave]` 显示对应表情动画 |
| **状态区** | 显示当前状态：`👂 聆听中...` / `💬 回复中...` / `😴 待机中` |
| **信息区** | 可选滚动显示学院最新动态、竞赛通知等 |

### 语音交互

| 参数 | 建议值 |
|------|--------|
| 唤醒词 | 「小芯」「嘿小芯」 |
| 语音识别超时 | 静默5秒后进入待机 |
| TTS音色 | 年轻男声，温暖亲切 |
| 回复最长时长 | 20秒（对应约60-80字中文） |

### 记忆系统

小芯在设备本地维护一个轻量级对话记忆：

| 记忆类型 | 示例 | 用途 |
|---------|------|------|
| 称呼 | 「叫我小明就行」 | 下次对话用名字称呼 |
| 专业 | 「我是电子信息的」 | 关联专业相关内容 |
| 兴趣 | 「我对机器人感兴趣」 | 推送相关竞赛信息 |
| 上次话题 | 「上次聊到智能车竞赛」 | 下次对话能接上 |

> 💡 **隐私说明**：所有记忆数据仅存储在设备本地，不上传云端。
